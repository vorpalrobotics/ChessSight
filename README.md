# ChessSight

A web app for training chess board vision and calculation discipline.

## What It Does

ChessSight presents targeted exercises to sharpen two core skills:

- **Board sight** — instantly recognize squares, diagonals, piece coverage, and tactical patterns without moving pieces
- **Training discipline** — build habits of candidate selection, pattern verification, and deliberate calculation

## Getting Started

No build tools required. Open `index.html` in a browser and start training.

## Stack

Vanilla JavaScript, HTML, CSS — no dependencies.
